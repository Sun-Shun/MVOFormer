<h1 align="center">MVOFormer: Flow-Semantic Transformer for Robust Monocular Visual Odometry</h1>

<p align="center">A Transformer-based monocular visual odometry framework integrating optical flow and semantic features for robust pose estimation.</p>

---

## 📖 Overview

MVOFormer proposes a novel architecture that leverages both motion (optical flow) and semantic cues through a Transformer-based design for monocular visual odometry. The framework achieves robust pose estimation by effectively fusing flow and semantic features, improving performance in challenging scenarios such as low-texture regions, dynamic objects, and illumination changes.
